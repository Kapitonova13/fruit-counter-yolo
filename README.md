# fruit-counter-yolo

# Система подсчета фруктов на основе YOLOv11

Веб-приложение для автоматического подсчета фруктов с использованием нейронных сетей компьютерного зрения.

## Функциональность
- Детекция и сегментация фруктов (яблоки, бананы, апельсины)
- Загрузка изображений через файлы и веб-камеру
- Визуализация результатов (маски, bounding boxes)
- Система отчетности (Excel)
- Сохранение истории обработки

## Технологический стек
- **Backend:** Flask (Python)
- **Нейронная сеть:** YOLOv11 (Ultralytics)
- **Обработка изображений:** OpenCV
- **Фронтенд:** HTML, CSS, JavaScript
- **База данных:** JSON (история запросов)
- **Отчеты:** pandas

## Быстрый старт

### 1. Клонирование репозитория
git clone https://github.com/ваш-логин/fruit-counter.git
cd fruit-counter
### 2. Установка зависимостей
pip install -r requirements.txt
### 3. Запуск приложения
python app.py
